    1. Open the Run dialog:                                 Win + R
    2. Type the following command in the Run box:           cmd
    3. Open Command Prompt as administrator:                ctrl + shift + enter
    4. Type the following command in the Command Prompt:    rundll32.exe sysdm.cpl,EditEnvironmentVariables
